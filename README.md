*** PhotoSphere - Galerie Photo Communautaire ***

Plateforme de partage de photos pour photographes amateurs et professionnels


- Description
Application web permettant aux photographes de partager leurs photos, créer des albums et interagir avec une communauté.
Fonctionnalités principales:

- Inscription/Connexion sécurisée
Upload de photos (quota BasicUser: 10/mois, ProUser: illimité)
Création d'albums publics/privés (ProUser uniquement pour privés)
Système de tags
Likes et commentaires
Dashboard statistiques (ProUser)
